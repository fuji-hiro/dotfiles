    What's this?

    overwrite .zshrc, .emacs and so on.

    Environment

    • RHEL 5.x (CentOS 5.x)
    • RHEL 6.x (Scientific Linux 6.x)

    How to use?

    $ cd
$ git clone git@github.com:tshirai/dotfiles.git
$ cd dotfiles

# edit main() and select files to be overwritten.
$ emacs bin/install_config.py

# for RHEL
$ sudo yum install texinfo
# for debian
$ sudo apt-get install texinfo

# install
$ bin/install_config.py -a
# update
$ bin/install_config.py

    Other information

    • Emacs package management: el-get
    • Environment Variables of Proxy: create ~/.proxyrc, then dotfiles/zshrc read it.
